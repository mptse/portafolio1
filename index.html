the-end

Sistema de Gestión de Cines

THE-END es una aplicación web full-stack diseñada para gestionar todas las operaciones de un cine: cartelera de películas, funciones, asignación de asientos y venta de tiquetes. El sistema implementa tres roles diferenciados — administrador, operario y cliente — cada uno con una experiencia y permisos adaptados a sus necesidades.

Visitar Sitio

Manjares Del campo

Manjares del Campo es un e-commerce especializado en productos agrícolas 100% naturales, diseñado para conectar productores del campo con clientes finales de forma simple y accesible. El flujo de compra está adaptado al contexto real del negocio: los pedidos se realizan a través de WhatsApp, y el administrador gestiona el ciclo completo del pedido — de realizado a pagado, enviado y entregado — con envío automático de factura al confirmar el pago. El panel de administración incluye un dashboard con gráficas de productos más vendidos, alertas automáticas de stock agotándose, gestión de productos y categorías, e inventario completo. Se construyo el backend con Django y la interfaz con HTML, CSS y JavaScript vanilla.

Stack

HTML CSS JavaScript-vanilla SQLite pythonanywhere
